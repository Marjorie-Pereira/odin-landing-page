# CSS Landing Page - Odin Project
In this project I could practice the main concepts I learnt in the CSS section of the Odin Project Foundations Course. The skills demonstrated here are:
- Linking the CSS file with the HTML using the `<link/>` tag;
- Setting colors for background and text;
- Setting a font for the project, in this case I used Google Fonts through the `<link>` tag;
- The box model, using `box-sizing:border-box;` and other properties such as padding, marging and border;
- The Flexbox foundations and how to position elements with `justify-content` and `align-items` and also `flex-direction`;

Images are from Unsplash, as shown in the code.
